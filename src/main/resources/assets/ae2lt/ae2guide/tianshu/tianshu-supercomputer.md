# Tianshu Supercomputer

The Tianshu Supercomputer is a fixed **7×7×7** multiblock crafting CPU. Its controller is centered on one lower outer edge. The two candidates at the center of the top and bottom faces must contain exactly one Tianshu Port; fill the unused candidate with casing.

The inner glass shell surrounds a **3×3×3** core chamber. Place exactly one main supercomputing unit in the chamber center and fill all 26 peripheral cells with storage or parallel units. At least one of each peripheral type is required.

Each storage unit contributes 64 MiB of crafting storage. Each parallel unit contributes 128 parallel operations before the main-unit multiplier. Parallelism is capped at 16,384.

| Main unit | Storage multiplier | Parallel multiplier |
|---|---:|---:|
| Baseline | ×1 | ×1 |
| Quantum | ×16 | ×2 |
| Overload | ×256 | ×4 |
| Multidimensional | Infinite | ×8 |

The port joins the AE2 network only while the structure is formed. Breaking a required block stops new work immediately; retained crafting state resumes safely when the structure is repaired and reconnects.
