# 天枢物质扭曲矩阵

**天枢物质扭曲矩阵**是一套接入 ME 网络的大型合成多方块。它直接执行样板仓内的样板，以可配置的发配、增幅与热控能力承担高吞吐量的基础材料与中间产物加工。

矩阵是**合成供应设备**，并非合成存储器。合成任务仍由 ME 合成 CPU 规划并提供原料；矩阵接收任务后完成合成，并将产物直接送回网络。

<SubPages />

## 基本工作流程

1. [搭建 7×11×7 结构](construction.md)，在中央放置一个主核心，并填满其余 80 个外围单元槽
2. 在至少一个样板仓内放入样板
3. 通过矩阵端口接入有能源的 ME 网络
4. 从 ME 终端发起合成任务；矩阵会作为该样板的合成供应设备参与任务

## 运行条件

* 结构必须完整成形，并且核心配置有效
* 必须恰好存在一个矩阵端口和一个主核心
* 至少安装一个样板仓
* 端口必须接入活动的 ME 网络；其待机功耗为 **8 AE/t**，结构成形后占用 **1 个频道**

结构本身不需要外部 FE，也不直接消耗闪电。
