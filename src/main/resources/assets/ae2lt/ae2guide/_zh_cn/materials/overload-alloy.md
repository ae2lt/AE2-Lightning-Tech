# 过载合金

**过载合金**是一种高级金属材料，广泛用于机器框架、过载控制器以及多种终末级组件。

## 加工流程

1. **过载合金粗胚** — 在工作台直接合成
2. **过载合金锭** — 成品合金锭，可在闪电模拟室中由粗胚精炼，也可在过载处理工厂中由金属与晶体类材料批量生产
3. **过载合金板** — 在闪电模拟室或过载处理工厂中由过载合金锭压制而成，是极限过载核心等顶级配方所需的材料

<RecipeFor id="ae2lt:overload_alloy_blank" />

过载合金锭可以由两种机器获得：

| 机器 | 输入 | 闪电 | 能量 | 产出 |
|------|------|------|------|------|
| 闪电模拟室 | `1` 过载合金粗胚 + `4` 过载水晶粉 | `1` 高压闪电 | `20,000` | `1` 过载合金锭 |
| 过载处理工厂 | `1` 铜块 + `1` 铁块 + `1` 金块 + `1` 紫水晶块 + `1` 末影之眼 + `1` 紫颂果 | `1` 高压闪电 | `100,000` | `27` 过载合金锭 |

## 物品一览

| 物品 | 作用 |
|------|------|
| 过载合金粗胚 | 中间产物 |
| 过载合金锭 | 成品合金锭，大多数过载机器的关键合成材料 |
| 过载合金板 | 压制板材，用于极限过载核心等高级配方 |
