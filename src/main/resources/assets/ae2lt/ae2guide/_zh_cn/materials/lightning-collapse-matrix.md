# 闪电坍缩矩阵

<ItemImage id="ae2lt:lightning_collapse_matrix" scale="2" float="left" />

**闪电坍缩矩阵**是 AE2 闪电科技中最核心的终末级组件之一。它在机器中不作为消耗品，而是作为**代偿与并行的催化元件**——在多种机器的矩阵槽中保留一枚矩阵，即可解锁该机器的高阶运行模式。

`使用高强度的合金将庞大的闪电压缩于其中，使其呈现出一种稳定的形态，其能量波与暴躁过载TNT的相反。`
`~~似乎可以互相中和的能量?~~`

## 获取方式

### 在闪电模拟室中合成

| 材料 | 数量 |
|------|------|
| 完美电鸣水晶 | 1 |
| 极限过载核心 | 16 |
| **极高压闪电** | 128 |
| 能量 | 50,000,000 AE |

由于成本较高，建议与过载处理工厂、闪电装配室的排产一并规划。

## 作为机器催化剂

闪电坍缩矩阵**不会**在加工中被消耗，但必须始终留在矩阵槽中：

* [闪电模拟室](../machines/lightning-simulation-chamber.md) / [闪电装配室](../machines/lightning-assembly-chamber.md) — 安装矩阵后，部分原本要求**极高压闪电**的配方可以用数倍量的**高压闪电**代偿
* [特斯拉线圈](../lightning/tesla-coil.md) — 极高压模式必须在槽中安装矩阵，否则无法从高压闪电升压到极高压
* [水晶催化器](../machines/crystal-catalyzer.md) — 水晶模式与粉化模式均可生效；安装矩阵后，并行后的单次产出再提升至 **4 倍**
* [过载处理工厂](../machines/overload-processing-factory.md) — 多矩阵并行：每多一枚矩阵解锁一档并行度（槽位最多 32 枚，默认每枚提供 8 并行）

## 快速安装与内存卡

手持闪电坍缩矩阵并对上述任意机器**潜行右键**，会按该机器自身的矩阵槽容量尽量装入：单矩阵机器装入 1 枚，过载处理工厂最多装到 32 枚，放不下的部分仍留在手中。

<ItemLink id="ae2:memory_card" /> 还会把已安装数量保存为机器模板的一部分。将模板读取到同类机器时，会恢复保存的数量：缺少的矩阵只从玩家主背包 / 快捷栏取出，多余的矩阵则退回背包（背包已满时掉落），并按目标机器当前的槽位上限裁剪。内存卡保存的是**期望数量而不是矩阵物品本身**，因此不会复制矩阵。

## 观测备忘

> 外场小组曾提交过一份额外的实验报告：矩阵在**掉落在世界中**时会出现一种非预期反应；反应被点燃的场景记在 [过载 TNT](../lightning/overload-tnt.md) 那一页。

产物是一枚不在记录中的元件，可以放入 <ItemLink id="ae2:drive" /> 中解析；这个元件只能被读取一次，不可写入。报告中指出，读取的样板分布极不均匀。多次实验后，实验小组总结出了一份特别的报告，其中写道："样本分布极不均匀，最稀有的那份不是元件，是一本还没写字的册子。"
