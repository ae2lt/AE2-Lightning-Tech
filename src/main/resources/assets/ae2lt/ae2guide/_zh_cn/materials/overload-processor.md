# 过载处理器

**过载处理器**是一种高级处理器，用于制作大多数过载机器和高级设备。其生产过程需要通过 AE2 压印器进行多步合成。

## 合成流程

1. 获取**过载压印模板**（通过闪电转化、闪电模拟室或过载处理工厂获得）
2. 在 AE2 压印器中使用过载压印模板与过载合金粗胚，压印出**未过载的电路板**
3. 将未过载的电路板放回压印器，加工为**过载电路板**
4. 将过载电路板与其他材料在 AE2 分子装配室中组合，即可获得**过载处理器**

## 物品一览

| 物品 | 作用 |
|------|------|
| 过载压印模板 | 在 AE2 压印器中使用的压印模板；通过闪电转化或机器加工获得 |
| 未过载的电路板 | 中间产物；用过载压印模板压印过载合金粗胚得到 |
| 过载电路板 | 成品电路板；由未过载的电路板在压印器中进一步加工得到 |
| 过载处理器 | 最终处理器，用于机器和设备的合成配方 |
