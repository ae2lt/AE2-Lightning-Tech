# 端口、任务与快速计算

## 天枢超算端口

结构成形后，<ItemLink id="ae2lt:tianshu_supercomputer_port" /> 会将整套多方块作为名为“天枢超算中心”的合成 CPU 发布到 ME 网络。端口的六个方向均可连接 ME 线缆，待机功耗为 **8 AE/t**，整套结构占用 **1 个频道**。

端口离线、ME 网络断电或结构解体时，天枢超算中心不会接受新任务。已经保存的任务状态不会因区块卸载而丢失；结构恢复并重新联网后可以继续执行。

## 共享容量与多任务

天枢超算中心与普通合成 CPU 的主要差异是可以同时保存并执行多个合成任务。

* 每个任务开始时，会从总合成容量中预留该计划所需的字节
* 只要剩余容量足够，新的任务可以继续分配给同一座天枢超算中心
* 任务完成或取消后，其预留容量会释放
* 核心配置显示的并行数可供每个活动任务使用

如果合成确认界面提示 CPU 容量不足，应等待现有任务释放容量，或增加存储超算单元、改用更高等级的主超算单元。

## 控制器界面

右击 <ItemLink id="ae2lt:tianshu_supercomputer_controller" /> 可以查看：

* 结构是否成形，以及未成形时优先检测到的问题
* 当前主超算单元等级
* 存储与并行超算单元数量
* 总合成容量、并行数及是否达到上限
* 自动搭建外壳与快速计算开关

更换核心舱单元会使结构暂时解体。若端口仍保存活动任务，新配置会在全部既有任务结束后生效；恢复原结构与网络连接可使任务继续执行。

## 快速计算

“快速计算”默认开启，用于缩短大型合成计划的计算时间。它只影响提交任务前的计划计算，不改变配方、原料需求、合成容量或任务执行速度。

快速计算无法处理部分计划或发生异常时，会自动回退到 AE2 的标准计算方式。若需要排查样板兼容性或对比计算结果，可以在控制器中将其关闭。

快速计算的启用条件按整个 ME 网络汇总。同一网络中只要仍有其他支持快速计算的活动合成 CPU 允许该功能，网络仍可能使用快速计算；需要完全停用时，应关闭其对应设置或使这些 CPU 离线。

## 故障排查

* **无法成形：** 检查固定外壳、玻璃与冷却单元，确认端口恰好一个、主超算单元位于中心、26 个外围槽已填满且两类外围单元均存在
* **结构方块齐全但仍未成形：** 清除展示结构中省略的必要空气区域；其中不能放置线缆、照明或装饰方块
* **自动搭建无反应：** 根据控制器提示清除阻挡，并确认所有材料位于玩家背包
* **合成确认界面中没有天枢超算中心：** 检查端口的线缆连接、ME 网络供电与结构成形状态
* **无法接受新任务：** 检查剩余合成容量；总容量可能已经由其他活动任务预留
* **并行数很高但加工速度未提高：** 检查样板供应器与加工设备数量、加工设备速度、原料供应以及网络传输能力
